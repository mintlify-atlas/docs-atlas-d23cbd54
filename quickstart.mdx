title: "Quick start"
description: "Get WebGoat running in 5 minutes using Docker."
---

The fastest way to run WebGoat is with Docker. You don't need to install Java or build anything from source — pull the image and you're ready to practice.

<Warning>
  WebGoat is intentionally insecure. Only run it on a machine that is disconnected from the internet or isolated on a private network. Never expose WebGoat on a public IP address.
</Warning>

## Prerequisites

- [Docker](https://docs.docker.com/get-docker/) installed and running on your machine.

## Steps

<Steps>
  <Step title="Install Docker">
    Download and install Docker Desktop for your operating system from [https://docs.docker.com/get-docker/](https://docs.docker.com/get-docker/).

    Verify Docker is running:

    ```bash
    docker --version
    ```
  </Step>
  <Step title="Run the WebGoat container">
    Pull and start WebGoat with a single command:

    ```bash
    docker run -it -p 127.0.0.1:8080:8080 -p 127.0.0.1:9090:9090 webgoat/webgoat
    ```

    This starts both WebGoat (port `8080`) and WebWolf (port `9090`). The container streams logs to your terminal — you can leave it running while you work through lessons.

    <Note>
      Some lessons depend on timezone-aware behavior. If you encounter time-related issues, pass your local timezone with the `-e TZ` flag:

      ```bash
      docker run -it -p 127.0.0.1:8080:8080 -p 127.0.0.1:9090:9090 -e TZ=America/New_York webgoat/webgoat
      ```
    </Note>
  </Step>
  <Step title="Open WebGoat in your browser">
    Once the container has started and you see log output, open your browser and navigate to:

    ```
    http://localhost:8080/WebGoat
    ```

    You'll be redirected to the login page.
  </Step>
  <Step title="Register an account">
    Click **Register new user** and create a username and password. WebGoat stores your lesson progress per user, so pick a name you'll remember.

    <Note>
      There is no email verification. Any username and password will work. This is a local-only application.
    </Note>
  </Step>
  <Step title="Start a lesson">
    Use the left-hand navigation to browse lessons by category. If you're new to WebGoat, start with the **Introduction** section to get oriented. Each lesson includes background reading followed by interactive assignments you complete against the live application.
  </Step>
</Steps>

## Other installation options

Docker is the recommended approach, but WebGoat also runs as a standalone JAR or can be built from source. See the [Installation](/installation) page for all options.
